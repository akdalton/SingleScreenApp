# SingleScreenApp
SingleScreenApp is basically an android based business application in which i have mentioned all the information related to business like Business name , address, contact no., email and website link.

## Technology Uses 
* Core Java
* XML

## Tools Required
* Android Studio IDE
* Java SDK
